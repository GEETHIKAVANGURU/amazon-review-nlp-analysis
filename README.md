# Identifying Hardware Failure Patterns in Amazon PC Reviews Using NLP

This project analyzes Amazon PC product reviews to identify common hardware issues mentioned in negative customer feedback.

Using Natural Language Processing (NLP) techniques, the project extracts patterns from customer complaints to understand which hardware components are most frequently associated with dissatisfaction.

## Dataset
Amazon PC product reviews dataset.

## Techniques Used
- Data exploration
- Text preprocessing
- Keyword extraction using CountVectorizer
- Topic modeling using LDA
- Word cloud visualization
- Hardware issue frequency analysis

## Key Findings
Analysis of negative reviews shows that hardware components such as screens, storage drives, and batteries appear frequently in customer complaints, suggesting that reliability of core device components plays a major role in negative user experiences.
